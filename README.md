# H.D.D-BounceProxy
 绝区零自动弹反系统
 
![img](./Docs/title.png)
## 特性
* 支持视觉检测/音频检测
* 自定义弹反招式
* 支持键盘/手柄
* 增加 GUI

## 使用

下载：`git clone https://github.com/Moxiner/H.D.D-BounceProxy.git`

环境安装：`./install.bat`

启动：`./run.bat`
